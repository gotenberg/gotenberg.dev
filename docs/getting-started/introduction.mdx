[Gotenberg](https://github.com/gotenberg/gotenberg) is a containerized API for seamless PDF conversion.

## Overview

* **Containerized & Self-Contained**<br />
Runs as a Docker image with all necessary dependencies bundled — no separate environment setup required.

* **Scalable & Distributed**<br />
Seamlessly integrates into distributed systems for efficient scaling.

* **Versatile Conversion Endpoints**<br />
Features robust *multipart/form-data* endpoints that convert various document formats to PDF.

* **Modern Protocol Support**<br/ >
Fully supports HTTP/2, including classic and H2C configurations.

* **Architecture & Flexibility**<br />
Gotenberg's modular design means each component is independently configurable.

* **Open Source**<br />
Gotenberg is open-source and free to use, making it an accessible solution for integrating PDF conversion into your projects.
