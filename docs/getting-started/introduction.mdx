[Gotenberg](https://github.com/gotenberg/gotenberg) is a containerized API that abstracts the complexity of PDF conversion.

It provides a `multipart/form-data` interface for interacting with powerful engines like **Chromium** and **LibreOffice**.
Instead of managing heavy dependencies, browser versions, or fonts in your own backend, simply send your files to Gotenberg and get a PDF in return.

## Quick Start

Spin up a container and convert a URL to PDF in two commands:

```bash
# 1. Start the container.
docker run --rm -p "3000:3000" gotenberg/gotenberg:8

# 2. Convert a URL to PDF.
curl \
--request POST http://localhost:3000/forms/chromium/convert/url \
--form url=https://sparksuite.github.io/simple-html-invoice-template/ \
-o invoice.pdf
```
