## Watermark & Stamp

### Native Watermark (HTML/CSS)

You can add a watermark directly in your HTML using CSS pseudo-elements. The watermark is rendered _behind_ the page content during conversion — no post-processing needed.

- **Process:** Happens _during_ the HTML-to-PDF conversion.
- **Performance:** High (no additional processing pass).
- **Flexibility:** Full control over styling, positioning, and content via CSS.

```css
body {
  position: relative;
}

body::before {
  content: "CONFIDENTIAL";
  position: fixed;
  inset: 0;
  z-index: -1;
  display: grid;
  justify-content: center;
  align-content: center;
  font-size: 160px;
  font-weight: bold;
  opacity: 0.08;
  transform: rotate(-45deg);
  pointer-events: none;
}
```

### Native Stamp (HTML/CSS)

A stamp works the same way but is rendered _on top of_ the page content using `::after` and a positive `z-index`.

```css
body::after {
  content: "APPROVED";
  position: fixed;
  top: 40px;
  right: 40px;
  z-index: 9999;
  padding: 10px 28px;
  border: 4px solid rgba(0, 128, 0, 0.6);
  border-radius: 8px;
  font-size: 36px;
  font-weight: bold;
  color: rgba(0, 128, 0, 0.6);
  transform: rotate(-12deg);
  pointer-events: none;
}
```
