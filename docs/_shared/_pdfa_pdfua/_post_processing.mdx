### Post-Processing (PDF Engines) {#post-processing-pdf-engines-pdfa-pdfua}

Use the `pdfa` and `pdfua` form fields to convert the PDF into a standardized format _after_ generation.

{props.engine === "PDF Engines Merge" ? "This option is only available for PDF Engines." : ""

- **Process:** Gotenberg {props.engine === "PDF Engines" ? "handles the PDF" : "generates the PDF"}, then re-processes it using LibreOffice (the only engine supporting these standards).
- **Performance:** Slower (requires a second conversion pass).
- **Result:** A compliant PDF/A (Archival) or PDF/UA (Universal Accessibility) document.

<ApiEndpoint
  curl={`
curl \\
--request ${props.method} http://localhost:3000${props.path} \\
${props.curlFormData}
--form pdfa=PDF/A-1b \\
--form pdfua=true \\
${props.curlOutput}
`}
/>
